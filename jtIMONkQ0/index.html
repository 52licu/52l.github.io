The server of "Naraka Bladepoint" failed and the player's game settlement screen crashed

2024-07-09 · 2 min read
Naraka Bladepoint

The latest news on July 9th: This afternoon, a large number of players reported that " Naraka Bladepoint " had game disconnections. The official Weibo account of "Naraka Bladepoint" released an announcement that a sudden network failure occurred in the server room, causing network fluctuations. Some players may have network connection problems such as disconnection and no response in queues.

　　NetEase 24 Studio said that the development team is currently working hard to repair the problem. "We sincerely apologize for the inconvenience caused to you and thank you for your patience and support for Naraka Bladepoint."

　　Some netizens said that the game "Naraka Bladepoint" prompted "Server is under maintenance, please try again later."

　　"Naraka Bladepoint" is a "battle royale" game under NetEase 24 Studio, which became a free game on July 14, 2023. According to the official Steam website, the peak number of online players of "Naraka Bladepoint" today was 297,000, and it has now dropped to 48,000 due to server maintenance.
